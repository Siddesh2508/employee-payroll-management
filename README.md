# Employee Payroll Management System

An Excel-based payroll management project for calculating employee salaries, overtime payments, and monthly compensation.

## Features

- Employee salary calculation
- Overtime hour tracking
- Overtime bonus calculation
- Weekly pay computation
- Monthly payroll summary
- Spreadsheet-based payroll management

## Technologies Used

- Microsoft Excel
- Spreadsheet formulas
- Payroll calculations

## Project Components

- Employee hourly wage tracking
- Work hours management
- Overtime calculations
- Total payroll computation

## Future Improvements

- Excel dashboard
- Automated payslip generation
- Database integration
- Python payroll automation
- Payroll analytics visualization
